#  DormWeb

**DormWeb** is a **digitalized hostel management system** designed to streamline and manage the operations of the **female hall of Islamic University of Technology**.  
It provides an efficient, user-friendly interface for both **students** and **administrators**, featuring a centralized database of all students‚Äô information and tools for managing essential hostel services.

---

##  Key Features

###  Admin Portal
Admins can log in to manage and monitor the dorm‚Äôs operations efficiently.

**Admin Features:**
-  **Search Student Database:** Access complete student information quickly.
-  **Cleaning Log:** Track and update cleaning schedules.
-  **Laundry Maintenance:** Manage laundry service records.
-  **Meal Tracking:** Monitor student meal records and preferences.
-  **Complaint Resolving:** View and resolve student complaints.
-  **Notice Submission:** Post announcements for students to view.

---

###  Student Portal
Students can log in to view notices, submit complaints, and stay updated on dorm activities.

**Student Features:**
-  **Dashboard:** View personalized hostel information at a glance.
-  **Gym Occupancy:** Check real-time gym usage.
-  **Study Room Occupancy:** Monitor availability of study spaces.
-  **Notice Update:** Stay informed about announcements from the admin.
-  **Complaint Submission:** Submit issues directly to the admin.
-  **Food Menu:** View the daily or weekly meal schedule.
-  **Update Cleaning Status:** Track and confirm room cleaning progress.

---

##  System Overview

**DormWeb** is built in **C**, focusing on modular programming and structured database management.  
It uses file handling to simulate a database and employs menu-driven interfaces for both **admin** and **student** users.
